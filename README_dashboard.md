# 📦 Logistics Dashboard — Excel BI

Интерактивный дашборд для анализа морской доставки товаров по США. Построен в Excel с использованием сводных таблиц, срезов и визуализаций.

---

## 🎯 Бизнес-задача

Компания занимается морской доставкой товаров по США и хочет понять:
- Какие службы доставки приносят наибольшую прибыль?
- В каких городах сосредоточен основной спрос?
- На какие дни и месяцы приходится пик заказов?
- Какова доля недоставленных и задержанных заказов?

---

## 📊 Данные

**Источник:** [US Logistics Performance Dataset — Kaggle](https://www.kaggle.com/datasets/shahriarkabir/us-logistics-performance-dataset)

| Параметр | Значение |
|---|---|
| Период | 2023 год |
| Строк | 2 000 |
| Столбцов | 11 |
| Тип данных | Логистика морской доставки США |

**Признаки:** ID клиента, склад отправки, город доставки, компания-перевозчик, дата отгрузки, дата доставки, вес, стоимость, расстояние, время доставки, статус заказа.

---

## 📐 Структура дашборда

### KPI-карточки
| Метрика | Описание |
|---|---|
| Общая прибыль | Сумма доходов по всем доставкам |
| Число доставок | Общее количество обработанных заказов |
| Среднее расстояние | Средняя дистанция доставки (мили) |
| Среднее время доставки | Средний срок от отгрузки до получения |

### Визуализации
| Блок | Тип | Аналитический вопрос |
|---|---|---|
| Динамика прибыли | Линейный график | Как меняется выручка по месяцам? |
| Распределение по городам | Гистограмма | Куда чаще всего доставляют? |
| Статус заказов | Круговая диаграмма | Какова доля потерянных и задержанных заказов? |

### Интерактивность
Дашборд управляется через **срезы** по:
- Компании-перевозчику
- Месяцу отгрузки

Все графики и KPI-карточки синхронно обновляются при выборе среза.

## 🛠️ Инструменты

- **Microsoft Excel** — основная среда

## 💡 Выводы

- Выручка неравномерно распределена по месяцам — есть выраженные пики спроса
- Несколько городов формируют основной объём заказов
- Доля задержанных и недоставленных заказов варьируется в зависимости от компании-перевозчика — это прямой сигнал для управленческих решений по выбору партнёра
- Срезы позволяют быстро сравнивать показатели разных компаний без дополнительных фильтров
